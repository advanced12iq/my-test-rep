# Сравнение времени достижения оптимального решения с tol=1e-6

## Обзор

В этом анализе сравниваются различные методы оптимизации по времени, необходимому для достижения оптимального решения с точностью tol=1e-6. Были протестированы следующие методы:

- Сорняковый метод оптимизации с адаптивным распространением
- Градиентный спуск
- Метод Нелдера-Мида
- Метод Пауэлла

Тестирование проводилось на трех классических функциях оптимизации:
- Функция Сферы (самая простая, унимодальная)
- Функция Розенброка (сложная, с извилистым глобальным минимумом)

## Выводы

1. **Наиболее точные результаты**:
   - На функции Сферы: Сорняковый метод адапт и Метод Пауэлла достигли целевой точности 1e-6
   - Ни один метод не достиг целевой точности 1e-6 на функциях Розенброка и Растригина

2. **Самые быстрые методы**:
   - Нелдер-Мид: показывает время 0 мс для всех функций (очень быстрый сходимость)
   - Градиентный спуск: 0-17 мс в зависимости от функции
   - Метод Пауэлла: 0-60 мс
   - Сорняковый метод адапт: 133-579 мс (самый медленный, но популяционный метод)

3. **Самые точные методы**:
   - На простой функции Сферы: Сорняковый метод адапт достиг идеального результата (0.000000)
   - На сложных функциях: все методы имеют проблемы с достижением 1e-6 точности

4. **Общая производительность**:
   - Для простых функций (Сфера): Сорняковый метод адапт показывает наилучшую точность
   - Для сложных функций (Розенброк, Растригин): все методы борются с достижением 1e-6 точности
   - Нелдер-Мид и Градиентный спуск работают быстрее, но не всегда достигают целевой точности
   - Сорняковый метод адапт более устойчив к сложным функциям, но требует больше времени

## Заключение

Сорняковый метод оптимизации с адаптивным распространением показывает хорошие результаты на простых функциях и имеет лучшую способность к точности, но требует больше времени. Для достижения tol=1e-6 на более сложных функциях может потребоваться увеличение максимального числа итераций или изменение параметров метода.
